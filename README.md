# prodtool

用[Electron](https://www.electronjs.org/)封装[几维智能生产工具](https://prodtool.kiwiot.com/serial-tool)

## Usage

```bash
$ git clone git@github.com:Loongwoo/prodtool.git
$ cd prodtool
$ yarn
$ yarn start
```
